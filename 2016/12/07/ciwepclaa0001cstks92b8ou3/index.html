ssh without password

1. Generate key

ssh-keygen -t rsa -C “peter@myhost” -f ~/ssh/myhost

2. Copy pub key to remote

ssh-copy-id -i ~/ssh/myhost.pub ${user}@${remote_host}

3. Login via private key

ssh -i ~/ssh/myhost ${user}@${remote_host}
